Group Management Commands

1. gpasswd -a user-name group-name -------- add user in group
2. gpasswd -d user-name group-name ------- remove user from group
3. gpasswd -A user-name group-name ------- set Administrative access to group
4. gpasswd -R user-name group-name ------- lock user
5. gpasswd -U user-name group-name ------- unlock user
6. groupadd ---- add new group
7. gpasswd ----- set passwd for group
8. gpasswd -r ---- remove password of group
9. gpasswd -M user1,user2,user3 group-name ----- add multiple users in group at once
10. groupdel group-name --- delelte group
